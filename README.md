# YFX0060 Learner

A simple learning tool for YFX0060 exam that checks your written answers with AI.

Written in pure Typescript and HTML. Used Tailwind and Express.js.

## Prerequisites

- Node
- npm

Install prerequisites yourself.

## Setup

Create `.env` file:

```bash
touch .env
```

Add your Anthropic API key there (Get it [here](https://console.anthropic.com/dashboard) - free for starters):

```
ANTHROPIC_API_KEY=your_api_key_here
```

## Starting

Use the `start.sh` script to start learning.

```bash
/start.sh
```

Open `http://localhost:3000` and voila.

## Happy learning!!
